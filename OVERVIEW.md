## 🖥️ Website Overview

A visually stunning, ultra-luxurious, high-converting website designed with a dark cinematic aesthetic and rich gold accents. The site is built to instantly make visitors feel:

> *"I trust this brand. This looks premium. I want to buy."*

### Structure

| Section | Description |
|---|---|
| **Hero** | Full-viewport cinematic background with aspirational headline, dual CTAs (Explore Collection + WhatsApp) |
| **Trust Strip** | 4 credibility badges + polished customer testimonial |
| **Collections** | 4-category product grid — Sneakers, Jewelry, Unisex Wears, Underwear |
| **Brand Manifesto** | "Why We Stand Apart" — 5 key selling points with editorial layout |
| **Featured Products** | Trending items with ₦ pricing, badges (Exclusive / Limited Stock), and direct ordering |
| **How to Order** | Simple 3-step visual flow: Browse → Order → Delivery |
| **Contact CTA** | Conversion-focused section with WhatsApp + phone call buttons |
| **Footer** | Brand info, navigation, collections, contact, social icons |

## 🎨 Design System

### Colors

| Token | Value | Usage |
|---|---|---|
| Primary Black | `#0A0A0A` | Main background |
| Deep Black | `#050505` | Footer / CTA backgrounds |
| Charcoal | `#1A1A1A` | Alternate section backgrounds |
| Surface | `#141414` | Card backgrounds |
| Rich Gold | `#C9A84C` | Primary accent, CTAs, highlights |
| Warm Gold | `#D4AF37` | Hover states |
| Text Primary | `#F5F0E8` | Headings, main text (warm white) |
| Text Secondary | `#A09888` | Body text, descriptions |
| Text Muted | `#6B6360` | Subtle labels |
| Accent Red | `#8B2500` | Urgency badges ("Limited Stock") |

### Typography

- **Headings:** Playfair Display — editorial luxury serif
- **Body:** Inter — clean modern sans-serif
- **Labels/Tags:** Inter uppercase with wide letter-spacing

### Animations

- Fade-in-up on scroll (staggered per grid)
- Gold shimmer on CTA hover
- Parallax effect on hero background
- Smooth scale transitions on product/category cards
- Scroll indicator pulse animation
- Navbar backdrop blur on scroll

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- npm

### Development

```bash
npm install       # Install dependencies
npm run dev       # Start dev server → http://localhost:5173/
```

### Production Build

```bash
npm run build     # Build for production → dist/
npm run preview   # Preview production build locally
```

## 📱 Features

### WhatsApp Integration
All ordering CTAs link directly to WhatsApp (`wa.me/2348104699214`) with pre-filled messages per product — the primary ordering channel for the Nigerian market.

### Mobile-First Responsive
- **480px** — Single column layout, stacked CTAs
- **768px** — 2-column grids, hamburger navigation
- **1024px** — Full grid layouts, desktop navigation
- **1440px** — Max-width container, optimal spacing

### SEO Optimized
- Descriptive `<title>` and `<meta description>`
- Open Graph tags for social sharing
- Semantic HTML5 elements
- Proper heading hierarchy (single `<h1>`)
- Unique IDs on all interactive elements

### Conversion Optimized
- CTAs placed strategically throughout every section
- Urgency badges ("Limited Stock", "Exclusive")
- Trust established early (testimonials + brand claims)
- Frictionless ordering via WhatsApp
